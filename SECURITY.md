# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in AirLock, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: **security@airlock.dev**

We will respond within 48 hours and work with you to understand and address the issue.

## Scope

AirLock is a security proxy designed to protect LLM API endpoints. The following are in scope:

- Bypass of prompt injection detection
- Bypass of sandbox analysis
- Request smuggling through the proxy
- Denial of service vectors
- Information disclosure

## Disclosure Policy

- We will acknowledge receipt of your report within 48 hours
- We will provide an estimated timeline for a fix within 7 days
- We will notify you when the fix is released
- We request that you do not disclose the vulnerability publicly until we have released a fix
